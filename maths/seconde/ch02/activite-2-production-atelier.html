Question 3 VAL

Quelle production hebdomadaire promettre à un nouveau client ? Justifier.

La valeur la plus fréquente (mode) est 18 chaises (33 %). C'est la production « typique ».

Pour être prudent : promettre 15-18 chaises/semaine. La meilleure semaine (25) est exceptionnelle, ne pas la prendre comme référence pour engager le client.

À retenir
  • Effectif : nombre d'occurrences. Fréquence : effectif / total.
  • Mode : valeur la plus fréquente. Bonne référence pour la « production typique ».
  • Engagement client : se baser sur le mode ou un quartile bas, pas sur le maximum exceptionnel.
