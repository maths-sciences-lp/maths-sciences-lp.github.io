Situation – lance-balle pour entraînement

Une balle lancée à plat suit l'équation : h(x) = −0,02 x² + 0,5 x où h = hauteur (m), x = distance horizontale (m).

Question 1 REA

Calculer h(0), h(5), h(12,5), h(25).

  • h(0) = 0 (départ)
  • h(5) = −0,02×25 + 2,5 = 2 m
  • h(12,5) = −0,02×156,25 + 6,25 = −3,125 + 6,25 = 3,125 m (sommet)
  • h(25) = −0,02×625 + 12,5 = −12,5 + 12,5 = 0 (atterrissage)

Question 2 ANA

Décrire l'allure du graphique.

C'est une parabole tournée vers le bas. La balle monte de 0 à 12,5 m horizontaux, atteint 3,125 m de hauteur, puis redescend pour atterrir à 25 m.

Coefficient négatif (-0,02) → branches vers le bas.

Question 3 VAL

À quelle distance la balle est-elle à 1,5 m de hauteur (montée et descente) ?

−0,02 x² + 0,5 x = 1,5 → 0,02 x² − 0,5 x + 1,5 = 0

Δ = 0,25 − 0,12 = 0,13 → √Δ ≈ 0,361

x = (0,5 ± 0,361)/(0,04) = 21,5 ou 3,5 m. Entre 3,5 m (montée) et 21,5 m (descente).

À retenir
  • Parabole : graphique d'une fonction f(x) = ax² + bx + c.
  • Si a < 0 : branches vers le bas, sommet = max.
  • 2 antécédents pour chaque image (sauf le sommet).
