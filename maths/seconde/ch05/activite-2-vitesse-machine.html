  • Résoudre ax + b = c

Situation – choisir le bon régime

Pour la coupe d'un panneau, le constructeur indique : la vitesse linéaire de coupe v (m/s) est liée à la vitesse de rotation N (tr/min) par v = 0,002 × N (lame de Ø 240 mm).

On veut une vitesse de coupe de v = 60 m/s pour le bois feuillu.

Problématique : Quelle vitesse N régler ?

Question 1 REA

Résoudre 0,002 × N = 60.

N = 60 / 0,002 = 30 000 tr/min

Régler la machine sur ce régime.

Question 2 REA

Pour un bois résineux, la vitesse de coupe doit être 80 m/s. Quelle nouvelle vitesse N ?

N = 80 / 0,002 = 40 000 tr/min.

Question 3 VAL

La machine a une limite à 25 000 tr/min. Que faire pour le feuillu ?

À 25 000 tr/min : v = 0,002 × 25 000 = 50 m/s. Insuffisant pour le feuillu (60 demandé).

Solutions : (a) lame plus grande (240 → 300 mm augmente v à régime égal) ; (b) accepter v = 50 m/s mais vitesse d'avance réduite ; (c) machine plus puissante.

À retenir
  • Équation 1er degré : ax = b → x = b/a (si a ≠ 0).
  • Application : tout réglage qui suit une loi proportionnelle (vitesse, dosage, prix).
