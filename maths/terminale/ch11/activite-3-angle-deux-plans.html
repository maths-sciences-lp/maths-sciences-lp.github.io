Activité 3 – Angle entre deux pans de toiture SITUATION PRO

Chapitre 11 – Produit scalaire | Terminale Bac Pro | ⏱ 30 min

Dernière mise à jour : 3 mai 2026, 09:00

Objectifs :
  • Calculer un angle entre 2 vecteurs avec le produit scalaire
  • Application concrète à la charpente

Situation – charpente à 4 pans

Sébastien, charpentier, conçoit une toiture à 4 pans. 2 pans adjacents se rencontrent en une arête. Pour fabriquer la pièce de jonction, il a besoin de connaître l'angle dièdre entre ces 2 pans.

Les 2 vecteurs normaux aux pans (perpendiculaires aux toits) sont :

  • Pan Sud : \(\vec{n_1}\) = (0 ; 0,5 ; 1) (incliné vers le Sud)
  • Pan Est : \(\vec{n_2}\) = (0,5 ; 0 ; 1) (incliné vers l'Est)

L'angle entre les pans = π − angle entre les normales.

Formule : cos(α) = \(\vec{n_1} \cdot \vec{n_2}\) / (||n_1|| × ||n_2||)

Problématique : Quel est l'angle entre les 2 pans ?

Question 1 REA

Calculer le produit scalaire \(\vec{n_1} \cdot \vec{n_2}\).

\(\vec{n_1} \cdot \vec{n_2}\) = 0×0,5 + 0,5×0 + 1×1 = 0 + 0 + 1 = 1

Question 2 REA

Calculer ||n_1|| et ||n_2||.

||n_1|| = √(0 + 0,25 + 1) = √1,25 ≈ 1,118

||n_2|| = √(0,25 + 0 + 1) = √1,25 ≈ 1,118

Question 3 REA

En déduire l'angle α entre les normales, puis l'angle dièdre β = 180° − α.

cos(α) = 1 / (1,118 × 1,118) = 1 / 1,25 = 0,800

α = arccos(0,800) ≈ 36,9° (angle entre les normales)

Angle dièdre entre les 2 pans : β = 180° − 36,9° ≈ 143°

Pour fabriquer la pièce d'arête, Sébastien sait maintenant qu'elle doit former un angle de 143° (un toit légèrement plat à la jonction).

À retenir
  • Produit scalaire 3D : u·v = u₁v₁ + u₂v₂ + u₃v₃.
  • Angle : cos(α) = (u·v) / (||u|| × ||v||).
  • Vecteurs orthogonaux : u·v = 0.
  • Application : géométrie 3D, charpentes, modélisation 3D, cristallographie.
