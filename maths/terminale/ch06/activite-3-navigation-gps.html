Chapitre 6 – Vecteurs | Terminale Bac Pro | ⏱ 30 min

Dernière mise à jour : 3 mai 2026, 09:00

Objectifs :
  • Utiliser les coordonnées de vecteurs en navigation
  • Calculer une distance et un cap

Situation – livraison à vélo cargo

Tom, livreur à vélo cargo, doit relier 3 clients dans le quartier (coordonnées en km dans un repère orthonormé centré sur le point de départ) :

  • A (3 ; 4) — boulangerie
  • B (−2 ; 5) — restaurant
  • C (1 ; −3) — bureau
Problématique : Quel parcours minimise la distance totale ?

Question 1 REA

Calculer la distance origine-A, origine-B, origine-C.

OA = √(3² + 4²) = √25 = 5 km

OB = √(4 + 25) = √29 ≈ 5,4 km

OC = √(1 + 9) = √10 ≈ 3,2 km

Question 2 REA

Calculer le vecteur \(\vec{AB}\) (déplacement de A à B). Sa norme.

\(\vec{AB}\) = (x_B − x_A ; y_B − y_A) = (−2 − 3 ; 5 − 4) = (−5 ; 1)

||\(\vec{AB}\)|| = √(25 + 1) = √26 ≈ 5,1 km

Question 3 ANA

Comparer 2 trajets : (a) Origine → A → B → C → Origine. (b) Origine → C → A → B → Origine. Quel est le plus court ?

Distances utiles :

  • OA = 5,0 ; OB = 5,4 ; OC = 3,2
  • AB = 5,1 ; BC = √(9 + 64) = √73 ≈ 8,5 ; AC = √(4 + 49) = √53 ≈ 7,3

Trajet (a) : 5,0 + 5,1 + 8,5 + 3,2 = 21,8 km

Trajet (b) : 3,2 + 7,3 + 5,1 + 5,4 = 21,0 km

Le trajet (b) est légèrement plus court (~ 4 % d'économie). Sur 100 livraisons/mois, ~ 80 km économisés.

À retenir
  • Vecteur entre 2 points : \(\vec{AB}\) = (x_B − x_A ; y_B − y_A).
  • Norme : distance = √(Δx² + Δy²) (Pythagore).
  • Optimisation de trajet : problème du voyageur de commerce. Tester plusieurs ordres pour trouver le plus court.
