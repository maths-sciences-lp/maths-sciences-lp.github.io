Activité 2 – Bénéfice maximal d'une production SITUATION PRO

Chapitre 4 – Polynômes degré 3 | Terminale Bac Pro | ⏱ 35 min

Dernière mise à jour : 3 mai 2026, 09:00

Objectifs :
  • Modéliser un bénéfice par un polynôme degré 3
  • Étudier ses variations pour trouver le maximum
  • Interpréter dans un contexte économique

Situation – atelier de meubles

Mathilde fabrique des chaises en bois. Pour x chaises produites par jour (avec 0 ≤ x ≤ 50) :

  • Recette R(x) = 80 x € (prix de vente unitaire 80 €)
  • Coût C(x) = 0,02 x³ − 0,5 x² + 30 x + 200 € (charges fixes + main d'œuvre + matières)

Le bénéfice est : B(x) = R(x) − C(x) = −0,02 x³ + 0,5 x² + 50 x − 200

Problématique : Combien de chaises Mathilde doit-elle produire par jour pour maximiser son bénéfice ?

Question 1 REA

Calculer B(x) pour x = 10, 20, 30, 40 chaises.

Le bénéfice augmente puis baisse. Maximum entre x = 40 et x = 50.

Question 2 REA

Calculer B'(x) (dérivée), puis résoudre B'(x) = 0 pour trouver le maximum exact.

B'(x) = −0,06 x² + x + 50

B'(x) = 0 → 0,06 x² − x − 50 = 0

Δ = 1 + 4 × 0,06 × 50 = 1 + 12 = 13

x = (1 ± √13) / (2 × 0,06) = (1 ± 3,606) / 0,12

x = 38,4 ou x = −21,7 (rejeté car < 0). Maximum à x ≈ 38 chaises/jour.

B(38) = −0,02×54 872 + 0,5×1 444 + 50×38 − 200 = −1 097 + 722 + 1 900 − 200 = 1 325 €

Question 3 VAL

Pourquoi le bénéfice baisse-t-il après 38 chaises ? Expliquer économiquement.

Au-delà de 38 chaises, le coût marginal (coût d'une chaise supplémentaire) dépasse les 80 € de prix de vente. Causes possibles :

  • Heures supplémentaires plus chères
  • Saturation des machines (perte de productivité)
  • Prix unitaires des matières premières qui grimpent (pénurie)

Conclusion : il vaut mieux produire moins mais mieux — l'optimum n'est pas le maximum de production. Recommandation : viser 35-40 chaises/jour.

À retenir
  • Polynômes degré 3 : permettent de modéliser des coûts à rendement décroissant.
  • Optimisation : max/min trouvé via la dérivée (B'(x) = 0).
  • En économie : l'optimum n'est jamais le maximum brut — on s'arrête où le coût marginal = revenu marginal.
