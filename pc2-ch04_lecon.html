Chapitre 4 – Composition chimique d'un liquide

2de Pro  |  Chimie

1. Corps pur et mélange

Définition
Corps pur : constitué d'une seule espèce chimique (ex : eau distillée H₂O).
Mélange : contient plusieurs espèces chimiques (ex : eau du robinet, jus de fruit).
Un mélange est homogène (solution) si on ne distingue pas les constituants ; hétérogène si on les distingue.

2. Solvant, soluté, solution

Solution aqueuse
Solvant : liquide en grande quantité (souvent l'eau).
Soluté : substance dissoute (solide, liquide ou gaz).
Solution = solvant + soluté(s). Solution aqueuse = solvant est l'eau.
Exemple
Eau sucrée : solvant = eau, soluté = sucre, solution = eau sucrée.

3. Concentration massique

Concentration massique
\[C_m = \frac{m_{soluté}}{V_{solution}}\]
  • \(m\) : masse du soluté (g)
  • \(V\) : volume de la solution (L)
  • \(C_m\) : concentration massique (g/L)
Méthode – Préparer une solution
Peser la masse de soluté voulue → verser dans une fiole jaugée → ajouter le solvant jusqu'au trait de jauge.
Exemple
Dissoudre 10 g de sel dans 500 mL d'eau : \(C_m = 10/0{,}5 = 20\) g/L

4. Dilution

Dilution
Diluer = ajouter du solvant pour diminuer la concentration.
Conservation de la masse de soluté : \[C_1 \times V_1 = C_2 \times V_2\]
Sécurité
Pour diluer un acide : toujours verser l'acide dans l'eau, jamais l'eau dans l'acide.
